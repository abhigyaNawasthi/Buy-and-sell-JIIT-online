
# Buy & Sell JIIT 🎒🛒

A vintage, earthy-themed peer-to-peer campus marketplace designed specifically for students at **Jaypee Institute of Information Technology (JIIT)**. This platform allows students to securely list, browse, and trade campus essentials, textbooks, and electronics using their official university credentials.

---

##  Project Status: Still in Development 🛠️
This platform is currently under active development. Core infrastructure is live, but several key features are being systematically built out and hardened.

### Current Features Complete:
*   **Hybrid Authentication Gate:** Secure student login via Magic Links and permanent passwords.
*   **Server-Side Domain Verification:** Hardened PostgreSQL database trigger preventing non-JIIT registrations.
*   **Earthy UI Dashboard:** Fully responsive layout styled with a custom vintage palette.
*   **Profile Matrix Backend:** Profile structures tracking student Name, Batch, Branch, Year, and Gender.

### Upcoming Milestones:
*   [ ] **"Sell an Item" Placement Form:** Product creation wizard supporting text listings.
*   [ ] **Supabase Storage Integration:** Direct image upload processing for physical items.
*   [ ] **User Item Management Hub:** Edit, delete, or mark listings as "Sold".
